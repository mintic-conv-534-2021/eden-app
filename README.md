# Eden App: Turista.
Aplicación para que brinda una experiencia digital a turistas en el eje cafetero.
En construcción. Hecho con Reactjs.

## Instalación

Abrir proyecto con visual studio code y ejecutar:


```react
npm i
npm run build
```

## Ejecución

```react
npm run start
```
